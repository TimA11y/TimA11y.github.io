Just In Time Status Messages

In these tests, when you press the buttons, an aria-live region will be added to the end of the form and then 1 ms afterwards text or an element will be added to the region. The message will disappear after a few seconds. The question is "Will a screen reader announce the message when it is repeated?"

Updating the Text Content Only

In this test, only the text contents of the aria-live region will be updated.

Updating Element

In this test, an element node will be added to the aria-live region.
